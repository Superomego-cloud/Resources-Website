# IOI TST 2026

This directory contains editorials for the problems from the IOI TST 2026 contest.
This contest served as a selection test to select Algeria's team in the International Olympiad in Informatics (IOI) 2026.

* [The Death Star](star.md)

* [Card Tricks](tricks.md)
